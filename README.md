# masyu

Masyu interface in python

## Execution

1. Create virtual environment
```bash
python -m venv env
```

2. Activate virtual environment
- Windows
```bash
env\Scripts\activate
```
- Linux
```bash
source env/bin/activate
```

3. Install requirements
```bash
pip install -r requirements.txt
```
